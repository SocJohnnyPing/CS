# 【1129 實習課隨堂筆記】
# Q1：price: $25.11 inc.VAT(取得價錢)
A ：price:\s+([^\s]+)
	• \s 空白
	• + 單一字元或群組出現至少一次

	• ()【括號】 把想找的相關字詞放入括號內，可依照括號裡的字元排序找到可能的結果。
	• EX：(sym) → sympathy、symbol、assym等

	• []【中括號】任意比對字串內的每個項目。
	•    EX：product[DEFG] :productD、productE、productF、productG
	• ^ : 搜尋規則前的「開頭」。意思代表｢非」
	• ^\s = 找非空白的
